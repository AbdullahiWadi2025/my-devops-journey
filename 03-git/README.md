# 03 - Git

Git is the industry standard version control system used to track changes, collaborate with teams, and manage code safely. This module focused on building a strong foundation in Git workflows that are used every day in software engineering, DevOps, and cloud engineering.

---

# What I Covered

- Git fundamentals
- Repository initialization (`git init`)
- Cloning repositories
- Staging and committing changes
- Viewing commit history
- Branching and branch management
- Merging branches
- Fast-forward vs three-way merges
- Merge conflict resolution
- Rebasing
- Stashing changes
- Reset, revert, and cherry-pick
- Working with remotes
- GitHub collaboration
- Pull Requests
- `.gitignore`
- Commit best practices
- Git security and SSH authentication
