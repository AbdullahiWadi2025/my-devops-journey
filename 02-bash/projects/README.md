# Bash Scripting

This repository contains Bash scripts that demonstrate core Linux system operations, automation tasks, and DevOps-style scripting workflows.

---

## 📂 Project Contents

### 1. calculator.sh
A command-line calculator that:
- Accepts two user inputs
- Performs addition, subtraction, multiplication, and division
- Handles division by zero safely

---

### 2. challenge3.sh
A file permission utility that:
- Checks if a file exists
- Displays read, write, and execute permissions
- Provides clear status messages for each check

---

### 3. challenge4.sh
A backup automation script that:
- Copies all `.txt` files from a source directory
- Creates timestamped backup archives
- Tracks and displays the number of files backed up

---

### 4. file_ops.sh
A file processing script that demonstrates:
- Reading files line by line
- Processing file content using loops
- Basic text handling operations

---

## 🧠 Skills Demonstrated

- Bash scripting fundamentals

- User input handling
- File permissions and validation
- Automation of system tasks
- Timestamp-based backup strategies
